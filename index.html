RedGradient

All about CSS gradients. Design quickly and create your own gradients
with a graphic pro-tools-grade editing system.

Visual Editor for Gradient.

Create sophisticated gradients with a mouse-based visual editor.

마우스 기반의 비주얼 에디터로 정교한 그라디언트를 생성할수 있습니다.

Visible Editing for GradientType
Resizing And Move Editing

Layer Group Base Edit.

Editing is possible through layers and layer groups. You can create various expressions using nested gradients.

레이어와 레이어그룹을 통한 편집이 가능합니다. 중첩된 그라디언트를 이용해 다양한 표현을 할수 있습니다.

Save/Load

You can save and load the project file you are working on.

작업중인 프로젝트파일을 저장하고 로드 할 수 있습니다.

User Preset Save/Load

User-based presets can be saved and recalled. Independently save/import border gradient, layer gradient, and group gradient Support.

사용자 기반의 프리셋을 저장하고 불러 올 수 있습니다. Border gradient, Layer gradient, Group gradient에 대한 독립적으로 저장/가져오기를 지원합니다.

Generate a single CSS Gradient output.
